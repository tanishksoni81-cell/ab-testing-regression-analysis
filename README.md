# ab-testing-regression-analysis
A/B Testing &amp; Regression Analysis project for marketing campaign optimization using Python and Power BI.

This project analyzes digital marketing campaign performance using A/B Testing and Regression Analysis techniques.

The objective is to identify the best-performing advertising platform and determine the key factors influencing campaign conversions.

## Tools & Technologies

- Python
- Pandas
- NumPy
- SciPy
- Statsmodels
- Scikit-learn
- Power BI
- Jupyter Notebook

## Project Objectives

- Compare Facebook and Google Ads performance
- Conduct statistical A/B testing
- Predict conversions using regression analysis
- Develop interactive Power BI dashboards
- Generate business recommendations

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. A/B Testing
5. Regression Analysis
6. Dashboard Development
7. Business Insights & Recommendations

## Key Insights

- Google Ads achieved higher conversion rates than Facebook Ads.
- Statistical testing confirmed significant platform differences.
- CTR and Clicks were the strongest predictors of conversions.
- Engagement metrics outperformed spending metrics.

## Recommendations

- Allocate more budget toward Google Ads campaigns.
- Improve CTR using better ad creatives.
- Monitor campaign KPIs weekly.
- Apply predictive analytics for future campaign planning.
